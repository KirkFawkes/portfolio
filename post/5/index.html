MortgageCalc

With MortgageCalc you can easily calculate monthly payments on mortgage rates and other loans.

When developing an application main requirement was cross-platform and use of PhoneGap. Application core created using Sencha Touch framework. Sending sms and email was implemented on native code for each of platform.

  • Languages:
    • JavaScript (Core)
    • Objective-C (iPhone)
    • Java (Android, Blackberry)
  • Platform: iOS 4.x+, Android 2.1+, Blackberry
  • Application Translations: English
  • Status: Completed and sent to the customer
  • Third-Party Libraries:
    • PhoneGap
    • Sencha Touch
