ForeclosureCanada

Bank foreclosures are properties that have defaulted on their mortgage payments and are re-possessed by the bank. In order to recover their loan as soon as possible, banks sell foreclosures often at massive discounts.

These incredible deals go fast, so investors or homebuyers with the most updated information have the buying advantage.

With this app, this advantage is now yours!

Find all types of realtor properties throughout canada. Alberta, British Columbia, Ontario, Manitoba, Saskatchewan, Nova Scotia. New Brunswick, Quebec and more.

Also find single-family homes, condos, townhomes, apartments, commercial properties, raw land and more!

Also find estate sales, probate sales, tax sales, motivated sellers, and many other types of properties that sell for below market value!

Develop project alone.

I had to develop an application for Foreclosure Search site.

Since target site doesn’t has API, my first task was analyze site code, MySQL database structure and develop API for getting list of real estate.

Also client has not provided a design for the application, so I took, as a basis, website design and similar client application written on some online builder tool.
