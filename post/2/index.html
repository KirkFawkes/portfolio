ZombieWrack

Disable all traps, throwing zombies into them.

Develop project alone.

First version was developed using custom 2d engine and Box2d. After some time, the game was completely rewritten in Objective-C and Cocos2d game engine. Also were used SpriteHelper for packing textures and LevelHelper for level editing.

After several months of development, project has been canceled.

  • Language: Objective-C
  • Platform: iOS 4+
  • Status: Canceled
  • Used Libraries:
    • Cocos2d
    • Box2D
  • Third party apps:
    • SpriteHeler - packing textures
    • LevelHelper - for WYSIWYG levele editing
