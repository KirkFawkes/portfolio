AutoFX2

С помощью AutoFX2 можно легко отправить заявку для получения кредита на покупку машины. Достаточно всего лишь ответить на несколько вопросов, сделать несколько фотографий и, когда заявка будет одобрена, машина ваша.

I developed the application according to the provided design. Вместе с back-end разработчиком был разработан API, позволяющий в фоне загружать результаты после каждого шага - это уменьшает время ожидания пользователя и, соответственно, улучшает user experience

  • Language: Objective-C
  • Platform: iOS 4.x+
  • Application Translations: English
  • Status: Completed and sent to the customer
  • Third-Party Libraries:
    • NimbusKit
    • AFNetworking
    • NimbusKit
